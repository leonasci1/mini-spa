Sou um Desenvolvedor Full-Stack e Aprendiz. Como idealizador do projeto pessoal musicblah!, venho construindo desde fevereiro de 2025 uma rede social musical. Nesse projeto, aprofundei conhecimentos em React, Vite, TypeScript, Node.js, Express, MongoDB e MySQL, assim como na resolução de desafios e bugs durante o desenvolvimento. Paralelamente, atuono Grupo SC Panpharma.

Possuo conhecimentos intermediários em Desenvolvimento Web Full-Stack, estudo desenvolvimento de bancos de dados com Java e SQL, e estudo programação em Python para analises de dados. Além disso estou estudando e praticando programação, em cursos com certificações da Amazon.

musicblah!

O processo de consumir música mudou radicalmente: antes, reuníamo‑nos em volta de um toca‑discos ou em festas para trocar impressões sobre novas faixas; hoje, em muitas casas, cada um ouve isolado em seu smartphone A missão da musicblah! é aproximar esses ouvintes, permitindo que compartilhem suas descobertas, publiquem resenhas e encontrem novos artistas guiados pelas recomendações de amigos.

Foi desenvolvido utilizando Vite, React e Typescript, MongoDB.

O projeto foi desenvolvido com o intuito de ser um projeto de estudo, onde o foco é a criação de uma aplicação que consome a API do Spotify.
